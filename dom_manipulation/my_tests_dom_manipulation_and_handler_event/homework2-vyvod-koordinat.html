Логика следующая - у нас есть текст на странице, при наведении на который, в консоли будет отображаться текущая координата мышки по x. Мы запилим прослушивание событий, т.е. у нас будет основная функция, которая будет слушать событие прогрузки страницы, внутри нее будет функция, которая будет искать элемент по тегу, слушать событие наведения мышки, и когда это событие произойдет, будет запускаться функция, которая будет выводить в консоль координату по x.
