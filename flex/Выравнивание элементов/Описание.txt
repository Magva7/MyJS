Как размещать элементы с помощью flexbox
Создавать расстояние между элементами можно по главной оси и по поперечной. Для того, чтобы создавать расстояние между элементами по главной оси, есть специальное CSS-свойство justify-content.

Посмотрим на примере. Без свойства justify-content наши элементы выглядят так:


Если указать justify-content: center, то элементы будут по центру:


Есть также значения justify-content: flex-start и justify-content: flex-end. Значение flex-start — когда элементы располагаются друг за другом и прижимаются к началу оси, а flex-end — к концу:



Значение justify-content: space-evenly размещает элементы так, чтобы между ними было равное расстояние:


Значение justify-content: space-between размещает первый элемент в начале оси, последний элемент — в конце оси, а затем делает равное расстояние между всеми элементами:


Есть еще и другие значения, которые выставляются для justify-content, но мы рассмотрели самые часто используемые. 

Итак, мы умеем распределять элементы по главной оси. Flexbox также позволяет распределять их по поперечной оси.

Как размещать элементы по поперечной оси?
Это можно сделать с помощью свойства align-items. Вот так будет выглядеть наш контейнер без выставленного align-items:


Как и у justify-content, элементы по вертикали можно отцентровать, используя align-items: center:


align-items: flex-start и align-items: flex-end тоже имеются:



Мы перечислили самые популярные значения для align-items. При этом значений space-between, space-evenly, как у justify-content, у align-items, нет. 

Опробуем знания на практике. Нам ведь нужно еще доделать шапку в лендинге. Посмотрим на ее текущее состояние:


Для нее нужно сделать 2 вещи:

Распределить логотип влево, а ссылку Контакты и кнопку Заказать вправо;
Отцентровать все по вертикали.
Для 1) к flex-контейнеру (тег header) нужно добавить:

justify-content: space-between;
Для 2) центруем по вертикали с помощью align-items:

align-items: center;
Таким образом, стили у header выглядят так:

header {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
